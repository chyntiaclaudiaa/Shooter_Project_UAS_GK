Proyek **Geo Strike** dikembangkan menggunakan **Unity** dan dijalankan melalui **Unity Editor**.  
Pastikan **Unity Hub** telah terpasang di perangkat Anda.

### Cara Menjalankan Proyek
1. Buka **Unity Hub**
2. Pilih menu **Open Project**
3. Arahkan ke folder proyek **Geo Strike**
4. Buka **Scene utama**
5. Tekan tombol **Play** pada Unity Editor

### Interaksi Pengguna
Pengguna dapat berinteraksi dengan game menggunakan **keyboard**, dengan kontrol sebagai berikut:
- **W** : Menggerakkan pesawat ke atas
- **S** : Menggerakkan pesawat ke bawah
- **Space** : Menembakkan peluru

Pemain memiliki **3 nyawa**. Setiap kali pesawat terkena musuh, nyawa akan berkurang dan ukuran pesawat akan mengecil sebagai indikator visual.  
Permainan akan **berakhir (Game Over)** jika nyawa habis.  
Pemain akan **menang (Victory)** apabila skor mencapai **100**.

### Contoh Input dan Output
- **Input** : Pengguna menekan tombol **Space**
- **Output** : Pesawat menembakkan peluru, musuh yang terkena peluru akan hancur, dan skor pemain bertambah
